# YAML ↔ JSON Converter

A free, fast, browser-based tool to convert between YAML and JSON. No sign-up, no tracking, no server — everything runs locally in your browser.

🔗 **Live tool:** [FRESHAVOCADO17.github.io/yaml-json-converter](https://FRESHAVOCADO17.github.io/yaml-json-converter)

---

## Features

- Convert YAML → JSON and JSON → YAML
- Clean, readable output with proper indentation
- Helpful error messages for invalid input
- Copy to clipboard in one click
- Works offline once loaded
- Dark mode support
- Mobile friendly

## Usage

Just open the link above. Paste your YAML or JSON on the left, click **Convert**, and copy the output.

Keyboard shortcut: `Ctrl+Enter` (or `Cmd+Enter` on Mac) to convert.

## Run locally

No build step needed. Just open `index.html` in your browser:

```bash
git clone https://github.com/FRESHAVOCADO17/yaml-json-converter.git
cd yaml-json-converter
open index.html
```
